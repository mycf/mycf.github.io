	`try-with-resources`语句是一种声明了一种或多种资源的try语句。资源是指在程序用完了之后必须要关闭的对象。`try-with-resources`语句保证了每个声明了的资源在语句结束的时候都会被关闭。任何实现了`java.lang.AutoCloseable`接口的对象，和实现了`java.io.Closeable`接口的对象，都可以当做资源使用。
